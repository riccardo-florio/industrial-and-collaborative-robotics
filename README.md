# Appunti di Industrial and Collaborative Robotics

## Obiettivo
Questo repository raccoglie in formato \LaTeX gli appunti del corso di "Industrial and Collaborative Robotics". Il materiale combina le slide del docente con le annotazioni personali redatte durante le lezioni.

## Metodo di lavoro
La trascrizione avviene attraverso un flusso che unisce ChatGPT e Overleaf: il testo generato viene riletto per assicurare coerenza e precisione, quindi i sorgenti \LaTeX vengono caricati in questa repository.

## Struttura
Il programma del corso è esteso e per facilitarne la consultazione gli appunti sono suddivisi in più parti, ognuna relativa a un modulo specifico. Ogni cartella del repository corrisponde a una di queste parti.

## Obiettivo finale
L'obiettivo a lungo termine è produrre una raccolta organica di appunti che, una volta riuniti, possano costituire un volume di riferimento per l'intero corso.
